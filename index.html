Sobre mí

Me apasiona crear interfaces rápidas, accesibles y hermosas. Esta plantilla está pensada para crecer contigo: componentes reutilizables, animaciones suaves y un sistema de diseño con variables CSS.

  • Responsive y accesible
  • Rendimiento primero
  • Sin dependencias pesadas

Características

Animaciones suaves con CSS + componentes listos para copiar/pegar.

Animaciones suaves

Entrada y transiciones con CSS. Sin jQuery ni librerías pesadas.

Diseño elegante

Tipografía limpia, gradientes sutiles, sombras suaves y glassmorphism.

Rendimiento

CSS mínimo, carga rápida y desplazamiento fluido.
